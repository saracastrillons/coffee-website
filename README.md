# Antioquia Coffee Website 

A modern and responsive website for a local coffee shop, built with clean and semantic HTML and CSS.  
Designed to showcase a business online through a professional landing page structure.



---

##  Features

- Responsive design for mobile, tablet, and desktop
- Hero section with branding message
- Navigation bar with links to key sections
- Featured products menu with pricing
- About section with story and mission
- Contact section with key communication channels
- Elegant and clean layout with modern UI

---

##  Technologies Used

| Technology | Purpose |
|-----------|---------|
| HTML5 | Website structure |
| CSS3 | Styling and layout |
| Git & GitHub | Version control |
| Netlify (optional) | Free hosting and deployment |
